# Climate Resilience Alert System

## Overview

Climate Resilience Alert System is a Machine Learning-based web application that predicts climate risks such as:

* Heatwave
* Flood
* Cyclone
* Safe Conditions

The system helps users understand potential climate threats based on weather parameters.

## Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* HTML
* CSS

## Features

* Climate risk prediction
* User-friendly web interface
* Machine Learning model using Random Forest
* Real-time prediction from user input

## How to Run

1. Install requirements

```
pip install -r requirements.txt
```

2. Train model

```
python train_model.py
```

3. Run application

```
python app.py
```

4. Open browser

```
http://127.0.0.1:5000
```
